## 도메인

### 발판
- [ ] 존재하거나, 존재하지 않는 두가지 상태만 가질 수 있다.


### Line(가로 줄)
- [ ] Line 에 속한 발판은 존재하는 상태를 연속으로 가질 수 없다.
- [ ] Line 은 사람 수보다 하나 적은 수만큼의 길이를 가진다.


### 사다리
- [ ] 사다리는 높이와 너비를 가진다.
- [ ] 사다리의 너비는 Line 의 길이와 동일해야 한다.
- [ ] 사다리는 높이만큼의 Line 개수를 갖는다.


## 입출력
- [ ] 게임에 참여할 사람들의 이름을 입력받는다.
  - [ ] 이름은 쉼표로 구분한다.
  - [ ] 이름은 최대 5글자까지 가능하다.
- [ ] 사다리 높이를 입력받는다.
- [ ] 사다리를 출력한다.
